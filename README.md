# ***********************************************************************************
1. **Iniciar**
   - Antes abrir el terminal y colocar `npm i --save-dev @types/node`
   - Despues ingresa en la carpeta de `frontend` y `backend`
   - Abre terminal en cada una y coloca `npm install`
   - Despues de instalar las dependencias coloca `npm start`
   - y listo
3. **Conectarse a MongoDB Atlas:**
   - Crear el archivo ".env" y colocar las credenciales y guardar el archivo dentro de `backend/src/`
